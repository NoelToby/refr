Detailed Description

Provides a set of named variables and their types, as well as the values for those variable.

N.B.: Primitive values are stored as strings, and are converted on demand to a concrete primitive type, one of {bool,int,double,string}. Because of this, this class only considers variables to have one of four types (even if their concrete types are more specific):

  • primitive
  • vector of a primitive type
  • Factory-constructible type
  • vector of a Factory-constructible type
