Detailed Description

This class is like a regular FeatureExtractor, but has been promoted to the executive level and thus wears fancypants.

More specifically, this class executes the extraction of a suite of feature extractors, specified at run-time via a set of string specifications, which are typically read from a file.

void reranker::ExecutiveFeatureExtractor::Init ( const string &  filename)

Initializes this instance from the feature extractor specifications in the specified file.

A feature extractor specification is a line that is of the form

 FeatureExtractorClassName(init_arg)

More formally, each line must have the form 

 <feature extractor classname> <lparen> <init str> <rparen>

where

<feature extractor classname> ::= the string passed to the REGISTER_NAMED macro by a FeatureExtractor implementation
<lparen> ::= a left parenthesis character, '('
<init str> ::= an initialization string to be passed to the newly-constructed FeatureExtractor’s FeatureExtractor::Init method
<rparen> ::= a right parenthesis character, ')'
Parameters:
filenamethe name of a file containing a set of feature extractor specifications, one per line
