virtual void reranker::PerceptronModelProtoReader::Init ( const string &  arg) [inline, virtual]

Initializes this instance with the contents of the argument string, which should either be the empty string, or else be the string "true" or "false", indicating whether to use “smart copying”.

“Smart copying” is when only one of the parameter vectors of a model is present in a message, and so this reader automatically copies the non-empty vector’s contents to the empty vector.
