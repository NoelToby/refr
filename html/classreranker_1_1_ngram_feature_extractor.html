virtual void reranker::NgramFeatureExtractor::Init ( const string &  arg) [inline, virtual]

Initializes this instance with the specified argument string, which is expected to contain the n-gram order for this feature extractor to extract and, optionally, a prefix string to be prepended each feature name produced.

This method is guaranteed to be invoked by one of the object creation methods of the Factory class just after construction.
