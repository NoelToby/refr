Outputs the symbol table to the specified output stream in a simple format, one symbol-to-index mapping per line:

<symbol> <tab> <index>

where

<symbol> ::= an indexed string symbol
<tab> ::= a tab character, '\t'
<index> ::= a unique integer for <symbol>
